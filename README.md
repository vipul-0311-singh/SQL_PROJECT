# 🎬 Netflix Movies & TV Shows SQL Analysis

## 📖 Introduction

With the rapid growth of streaming platforms, **Netflix** has become one of the world's leading providers of digital entertainment. This project analyzes the **Netflix Movies and TV Shows Dataset** using **MySQL** to extract meaningful insights, identify content trends, and demonstrate practical SQL skills through data cleaning, transformation, and analysis.

---

## 🎯 Project Objectives

The main objectives of this project are:

- Analyze the distribution of Movies and TV Shows.
- Identify the top contributing countries.
- Explore content growth and release trends over the years.
- Discover the most common genres on Netflix.
- Perform data cleaning and transformation using SQL.
- Practice real-world SQL querying techniques.

## 🔍 Key Insights

- 🎥 **Content Mix:** Movies significantly outnumber TV Shows.
- 📈 **Recent Growth:** Netflix content increased rapidly after 2020.
- 🌍 **Top Contributing Countries:** United States, India, and the United Kingdom contribute the most content.
- 🔞 **Popular Ratings:** Most titles are rated **TV-MA** and **TV-14**, indicating a focus on mature audiences.
- ⏱️ **Long Movies:** Many movies exceed 120 minutes, reflecting an emphasis on premium cinematic content.
- 🎬 **Top Directors:** A small number of directors account for a large share of Netflix titles.
- ⚠️ **Missing Data:** Several records lack director information, highlighting data quality issues.
- 🎭 **Popular Genres:** Drama and International content are the most common genres.
- 🗄️ **Database Organization:** Views and structured SQL queries improve readability and maintainability.
